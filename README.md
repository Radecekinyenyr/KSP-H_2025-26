   - úlohy 2,3,S jsou teoretické, tudíž je hodnocen hlavně slovní popis
   - nicméně pro všechny úlohy jsem napsal řešení přímo v kódu, abych měl jistotu
Vypracování 1. série, ačkoliv ještě není plně obodovaná, bych zhodnotil středně. Většina teoretických úloh si myslím, že se mi povedla (řešení zadání). Avšak slovní popisy těchto algoritmů např. pro úlohu č. 3, byly kostrbaté a neuspořádané. Z veliké části to bylo už z vyčerpání a také faktem, že jsem teoretické úlohy sepisoval prvně.
Odhad celkového počtu strávených hodin na první sérii je okolo 50-70 hodin.
